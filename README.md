# 🧰 慧盈AI工具庫

美國專利 USPTO 相關作業的 Streamlit 工具集，部署於 Streamlit Cloud。

---

## 工具清單

### 1. 📄 下載美國OA中的US前案（完整版）

**用途：** 上傳 USPTO Office Action PDF → 自動辨識並擷取 PTO-892 表格中的美國專利號碼清單 → 可手動修改清單 → 批次下載所有引用專利的 PDF。

**使用步驟：**
1. 上傳含有 PTO-892（Notice of References Cited）的 Office Action PDF
2. 點擊「擷取美國專利號碼清單」
3. 在可編輯文字框中確認或修改清單（可新增、刪除、修正號碼）
4. 點擊「確認清單，進入下載步驟」
5. 調整下載間隔與逾時設定後，點擊「下載」
6. 下載過程中可隨時按「⛔ 取消」中斷，已下載的部分仍可打包下載

**特色：**
- 含文字層的 PDF：使用 Regex 直接擷取，**免 API 費用**
- 純掃描圖像 PDF：自動切換 Claude Vision OCR 辨識
- 支援多頁 PTO-892（Page 1 of 2 等）
- 擷取結果可手動修正（OCR 錯誤、新增遺漏的號碼）
- 下載過程可隨時取消

---

### 2. ✍️ 手動輸入專利號來下載PDF

**用途：** 直接輸入美國專利號碼清單，批次下載 PDF。跳過 OA 上傳與 OCR 擷取步驟，適合已知專利號碼時快速使用。

**使用步驟：**
1. 在文字框中輸入專利號碼（每行一個）
2. 調整下載間隔與逾時設定
3. 點擊「🚀 開始下載」
4. 下載過程中可隨時按「⛔ 取消」中斷

**支援的號碼格式（自動清理）：**
- 一般實用專利：`7654321`
- 設計專利：`D959480`
- 再發專利：`RE45678`
- 植物專利：`PP12345`
- 公開申請案：`20220242717`
- 含前綴格式：`US7654321`、`US-7,654,321-B2`（自動去除前綴與 kind code）

## 下載來源與回退機制

每件專利依以下順序嘗試下載，前一個失敗才嘗試下一個：

1. **USPTO 官方** — `image-ppubs.uspto.gov`（優先）
2. **Google Patents** — `patents.google.com`
3. **FreePatentsOnline** — `freepatentsonline.com`
4. **Pat2PDF** — `pat2pdf.org`

## 環境設定

### Anthropic API Key

工具一與工具三的 Claude Vision OCR 功能需要 Anthropic API Key。

**方法 A（推薦）：** 在 Streamlit Cloud Secrets 加入：
```
ANTHROPIC_API_KEY = "sk-ant-..."
```

**方法 B：** 在左側邊欄手動輸入（每次重新整理頁面需重新輸入）。

> 工具二（手動輸入）不需要 API Key。

### 相依套件

```
requests
pymupdf        # fitz，用於 PDF 解析與頁面轉圖
```

## 注意事項

- 本工具僅供個人研究使用，請遵守各資料來源的服務條款
- USPTO 對雲端伺服器 IP 可能有頻率限制，若遭封鎖可調高「下載間隔」
- 下載輸出為 ZIP 壓縮檔，檔名格式為 `序號-專利號碼.pdf`，例如 `01-7654321.pdf`
